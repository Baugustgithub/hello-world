Procurement Method Selection Guide
Follow this guide to determine the most appropriate procurement approach for your needs.

Overview of VCU Procurement Methods

Up to $10,000
Delegated Authority

Departments can make purchases directly through RealSource or through use of a Pcard. Look for existing contracts first to maximize value. Follows a self-service model with required documentation.

Use appropriate payment method based on supplier requirements.

$10,000 to $200,000
Small Purchase Methods
  • BVA: Best Value Acquisition (multiple criteria)
  • RFQ: Request for Quote (lowest price)

Timeline: 5-60+ days based on complexity

Over $200,000
Formal Process
  • RFP: Request for Proposal (best overall value)
  • IFB: Invitation for Bid (lowest price)

Timeline: 45-125+ days based on complexity

Alternative Methods

Cooperative Procurement

Use existing contracts established by other public entities to streamline purchasing.

Timeline: 5-10+ business days

Sole Source Procurement

When a product/service is only available from one source. Requires justification.

Timeline: 5-15+ business days

Emergency Procurement

For urgent purchases where delay would cause serious harm.

Timeline: 1-5 business days

Exceptions to Competition

Specific categories exempt from competition requirements (e.g., educational materials, legal services).

Timeline: 5-15+ business days

First Step: Check for Existing Contracts
Before proceeding with a new solicitation, you should always first check if there are existing VCU contracts or cooperative contracts that can be utilized.
Check for Existing Contracts First
Before proceeding with a new solicitation, check if there are existing VCU contracts or cooperative contracts that can be utilized.
Why Check for Existing Contracts?
Using existing contracts can save significant time and resources. It eliminates the need for a new competitive solicitation process.
